 =====================================================
  CELLMAKER :: A toolkit for cellular level radiation dosimetry
 =====================================================

  [ ORIGIN ]
  Developed by RARiS Nuclear Lab
  Tohoku University, Japan

  [ SYNOPSIS ]
  An open-source utility for creating complex cell 
  array models used in Monte Carlo radiation 
  transport studies.

  [ BUILD ]
  $ make

  [ USAGE ]
  $ ./PHITSCellMaker

  [ STATUS ]
  Active // Open Source

  [USER MANUAL]
 1. Launch: Run ./PHITSCellMaker to open the GUI.
 2. Configure: Set your cell geometry, materials, and radiation parameters within the interface.
 3. Generate: Click "Generate" to create the PHITS input file. It is automatically saved to the program_folder/ directory.
 4. Execute: Run the simulation directly by clicking "Run PHITS" in the GUI.
 5. Analyze: Retrieve your output logs and tallies from program_folder/.
